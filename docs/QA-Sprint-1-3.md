- [ ] Submit Reading attempt.
- [ ] Submit the same Reading question twice and confirm attempts are not overwritten.
- [ ] Submit Listening wrong answer with one error reason.
- [ ] Submit Writing task.
- [ ] Submit Speaking task.
- [ ] Confirm practice session history is stored.
- [ ] View Parent Feedback and confirm active session can end.

## Analytics

- [ ] Today summary uses real local attempts.

- [ ] Accuracy by paper updates after objective attempts.
- [ ] Reading part performance updates after Reading attempts.

- [ ] Top weak diagnosis tags are from wrong attempts.
- [ ] Writing task point completion updates.
- [ ] Speaking output length trend updates.

## Wrong Answer Review

- [ ] Wrong objective attempt appears.
- [ ] Question title, paper, part, topic are visible.
- [ ] Selected answer and correct answer are visible.

## Parent Feedback

- [ ] Uses five-section structure.
- [ ] Mentions today attempts when available.
- [ ] Mentions recent weak paper or insufficient data.
- [ ] Mentions listening error reason when available.
- [ ] Mentions writing or speaking trend when available.
